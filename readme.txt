Drag and drop wordpress visual theme designer framework, featuring integrated LessCSS support, literally allows you to customize any layout in wordpress template hierarchy, simplified widget and dynamic sidebar administration and ability to extend through wp plugin api .

== Description ==

Layout Engine simplifies the process of features rich theme building using LessCSS, provides new ways to define default layout and possiblity to override layout by end users using simple drag and drop interface.

== Installation ==

You can download and install Layout Engine using the built in WordPress plugin installer. If you download manually, make sure it is uploaded to "/wp-content/plugins/layout-engine/".

Activate Layout Engine in the "Plugins" admin panel using the "Activate" link.

== Frequently Asked Questions ==

= Can I use my existing WordPress theme? =

Of course! Layout engine requires very minimum effort to add features offered by plugin. 
Please consider downloading <a href="http://simpleux.co.uk/plugins/wordpress/layout-engine/">getting started theme</a> to understand how easily you can convert <a href="http://wordpress.org/extend/themes/twentyeleven">twenty eleven</a> into layout engine <a href="http://simpleux.co.uk/plugins/wordpress/layout-engine/">twenty eleven theme</a>.

2. **WP Customizer** - Automatically create wp customizer settings.
